# Binance Futures Trading Bot (Testnet)

Python CLI application to place Market and Limit orders on Binance Futures Testnet (USDT-M).

## Features
- Market & Limit orders
- BUY / SELL support
- Command Line Interface (CLI)
- Logging
- Error handling
- Testnet only (no real money)

## Setup

Install required libraries:

```bash
pip install python-binance python-dotenv
